# AOI_finder
## Usage
利用百度地图能力，根据POI检索可能存在的对应AOI。web在线即用。

## Tech stack
- nuxt3, vue3, ts
- mapbox
- unocss

## Based on
- baidu map api: 
  - place-suggestions
  - get-aoi-by-uid
